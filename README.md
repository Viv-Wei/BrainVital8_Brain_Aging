# BrainVital8_Brain_Aging

***Summary of the codes used in the study***


## Core codes

1. `BrainVital8-Alignment.ipynb`: Alignment Summary Table for BrainVital8.
2. `BrainVital8-Competing_risk.R`: Fine-Gray models across cohorts for BrainVital8’s association with dementia risk.
3. `BrainVital8-GeneCell_specificity_score.ipynb`: Gene Specificity Score Analysis.
4. `BrainVital8-Figure1b-BAG_calculation.ipynb`: Brain Age Gap (BAG) Pipeline.
5. `BrainVital8-Figure1c-Manhattan_plot.ipynb`: Manhattan Plot: Full-scale GLM + BH FDR.
6. `BrainVital8-Figure1d-heatmap_plot.ipynb`: Brain Aging Analysis Pipeline (Heatmap Visualization).
7. `BrainVital8-Figure2c-SEM.R`: BrainVital8 Second-Order Structural Equation Model.
8. `BrainVital8-Figure4-mega_analysis.ipynb`: Stratified Cox models for BrainVital8’s dementia risk across cohorts, ages, sexes, and BMI.
9. `BrainVital8-Figure5-6-AUC_plot.ipynb`: Continuous AUC(t) Curves for BrainVital8.
10. `BrainVital8-Figure5-6-calibration_plot.ipynb`: Calibration plots, forest plots, and summary table across 6 cohorts.
11. `BrainVital8-Figure5-6-DCA_netbenefit.ipynb`: BrainVital8 DCA - Standardized Net Benefit.
12. `BrainVital8-Figure5-6-iAUC.ipynb`: Identifies preclinical dementia markers by iAUC, retention, timing, and sensitivity in cohorts.
13. `BrainVital8-Figure8a-wordcloud.ipynb`: Generates color-coded wordcloud of SNP categories using Nature palette.
14. `BrainVital8-Figure8b-MR_analysis.R`: Mendelian Randomization Analysis.
15. `BrainVital8-Figure8c-gsmap_mouse_Brain.ipynb`: gsMap Mouse Whole-Brain MERFISH Mapping Pipeline.
16. `BrainVItal8-Figure8d-gsmap_human_Brain.ipynb`: gsMap Three-Tier Spatial Transcriptomics Mapping Pipeline.
17. `BrainVital8-Figure8e-f-gsmap_single_cell.ipynb`: Full gsMap pipeline: GWAS QC, scRNA prep, LDSC, mapping, causal genes, and visualization.
18. `BrainVital8-Figure9-10-colocalization.ipynb`: Performs colocalization between GWAS and eQTL across gene–cell pairs using coloc.abf.
19. `BrainVital8-Figure9-10-csMR.ipynb`: Cell-type-specific Mendelian Randomization (csMR) Pipeline.
20. `BrainVital8-Figure9a-circular_volcano.ipynb`: Creates circular volcano plots with FDR-mapped colors, effect sizes, and cell-type bands.
